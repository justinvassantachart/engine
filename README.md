# runtime

WebAssembly compiler and runtime

## Getting Started

1. Install Rust

```sh
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

2. Add `~/.cargo/bin` to your `PATH` (e.g. in `~/.zshrc`).

```sh
export PATH="$PATH:~/.cargo/bin"
```

3. Install `wasm-bindgen-cli`

```sh
cargo install wasm-pack
```
